Parspas getway for wordpress.

== Description ==
Very easy and quick connection to the online Parspal payment (Simple version)

== Installation ==
1. Upload `parspal` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto setting page and complete the configuration.
4. To display payment form, using `<?php parspal_form(); ?>` function in theme or insert [parspal] shortcode in widget text or page/post.
